Smartgrid Lab 관리자 로그인

콘텐츠 편집기는 관리자 계정으로 로그인한 뒤 사용할 수 있습니다.

Smartgrid Lab 콘텐츠 편집기

뉴스, 멤버, 논문, 과제 데이터를 수정한 뒤 `data/lab-data.js`로 저장합니다.

권장 흐름
1. 항목을 수정합니다.
2. `파일로 저장`을 누릅니다.
3. `data/lab-data.js`를 선택해 덮어씁니다.
4. 홈페이지를 새로고침합니다.
5. `publish-site.cmd`를 실행해 GitHub에 올립니다.
GitHub 업로드
편집기는 Git을 직접 실행할 수 없습니다. 저장 후 폴더에서 `publish-site.cmd`를 더블클릭하면 변경 목록 확인, 커밋, 푸시를 순서대로 진행합니다.
현재 `data/lab-data.js`에서 데이터를 읽었습니다.
