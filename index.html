Research Direction

전력망 전환기의 안정도, 유연성, 운영 기술을 연구합니다.

Smartgrid Laboratory는 전력전자 기반 송전·계통연계, 측정 데이터 기반 계통 감시, 재생에너지·저장장치 통합 운영을 중심으로 미래 전력망의 핵심 문제를 다룹니다.

01 HVDC / FACTS / MMC

전력전자 기반 설비의 모델링, 해석, 제어 전략

02 Monitoring / Dynamics

PMU, 동적 부하모델, 전력품질 기반 계통 감시

03 Renewable / ESS / EMS

재생에너지와 저장장치가 결합된 저탄소 전력망 운영

Research Pipeline Modeling → Analysis → Control → Validation

수치 모델과 실계통 데이터를 연결해 운영 가능한 전력망 기술로 확장합니다.
