# 2026-06-13 — 국내 공대 전력/EE 동료 연구실 실사 (일반론 교정)

세 AI 토론이 실제 동료 연구실을 안 보고 일반론(화려한 히어로·검색·대시보드)을 냈다는 지적에 따라, 같은 분야 실제 peer 홈페이지를 직접 확인.

## 실사한 곳
- **연세대 PCSL — 전력·제어시스템 연구실** (https://pcsl.yonsei.ac.kr/) — 같은 학교·같은 분야 직속 형제 랩
- **서울대 PESL — 전력전자시스템연구실** (https://pesl.snu.ac.kr/)
- 고려대 차세대전력망 연구실(장길수, http://pel.korea.ac.kr/) — self-signed 인증서로 접근 차단, 미확인

## 실제 패턴 (확인된 것)
- **홈에 큰 히어로 이미지·애니메이션 없음.** 로고 + **최신 공지(News/Notice)**가 메인 — 즉 뉴스/공지 중심 (PCSL, PESL 공통)
- **Publications·Patents를 International / Domestic으로 분리** (PCSL). 학회까지 포함.
- **Board 메뉴**: 공지(Notice) + 자료실(Data Room) + **포토갤러리** — 실제 행사 사진(포럼·송년회·창립 N주년). PESL은 Board 연도별 아카이브(2019~2026).
- **Honors/Awards** 전용 메뉴(PCSL).
- 멤버는 전용 페이지에 실제 사진.
- 전반적으로 **정보/게시판 중심의 실용적 구성** — 스타트업식 비주얼 히어로가 아님.

## 일반론 교정 (peer 증거 기준)
- 화려한 히어로/애니메이션 → peer는 안 씀. 우리 계통도 히어로는 유지하되 **홈에 뉴스가 빠진 게 진짜 결함**이었음.
- Fuse.js 검색·데이터 대시보드/로고클라우드 → **어떤 peer도 안 함.** 연도그룹 + International/Domestic 분류면 충분. (블라인드 심판 2명도 대시보드를 "최약체"로 지목)
- 논문 = 국제저널만(62) → peer는 **국내 저널·학회까지** 분류. 추후 보완 후보.

## 우리 사이트에 적용 (이 날)
- **홈에 Latest News 스트립 추가** (최신 소식 3개 + 모든 소식 보기) — peer 표준 반영
- 호버 드롭다운 메뉴에 **터치 탭 토글** 추가 (모바일/태블릿 접근성)
- 폐기 확정: 데이터 대시보드/로고클라우드/과한 검색

## 추후 (데이터·사진 필요)
- 멤버 실제 사진 + 단체사진 + 행사 포토갤러리
- Publications에 Domestic/Conference 섹션
- Honors/Awards (실제 수상 생기면 — placeholder는 금지)
