사이트 코드와 무관하며, 이 폴더의 파일은 배포에 영향을 주지 않습니다.

## 운영 방식

- AI(Claude Code / Codex)가 주기적으로 새 레퍼런스를 조사해서 날짜별 파일로 추가합니다.
- 파일명: `YYYY-MM-DD-주제.md` (예: `2026-06-12-trendy-lab-websites.md`)
- 새 파일을 추가하기 전에 **기존 파일들을 먼저 읽고 중복 사이트는 제외**합니다.
- 실행용 프롬프트는 [PROMPT.md](PROMPT.md)에 있습니다 — AI에게 그대로 붙여넣으면 됩니다.

## 우리가 찾는 것

"학생 많고 유명하고 디자인 정돈된 대형 연구실/연구기관" 감성:

1. 실사 사진(단체사진, 캠퍼스, 장비) 중심의 히어로
2. 멤버/논문/과제 숫자 카운터
3. 날짜 붙은 뉴스·Activity 피드가 메인의 큰 비중
4. 대학원생 모집 공고 전면 배치
5. 소속/협력기관 로고 행
6. 화이트 베이스 + 포인트 컬러 (우리는 블루)
